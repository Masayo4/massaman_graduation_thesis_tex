Statical Shape Model
(パラメータ p_scale の話を使うための調査)

モデル作成にはshape model がとても重要
眉毛, 目などのディテクションは大半大丈夫
顔の向きだったり,表情だったりが大事

68点のモデルなど普遍的に人間の顔パーツの動きをトラッキングできるようにはなっている
顔の輪郭と, 感情認知に置いて重要なパーツのモデルをtadasは使用している
(重要なパーツ 眉毛, 唇, 鼻, 目)


非剛体変形の線形モデル　(表情は変わるので, 非剛体)
=> 点群統計モデルというらしい(PMD)

X = [X1,X2,...,Xn,Y1,Y2,...,Yn,Z1,Z2,...Zn]^T. 4.5
n = 顔の特徴点の数, x,y,zの座標点

X = Xバー +Φq. 4.6
xバー = 顔の形を示す, Φ(ファイ)　= n*m マトリックス, m次元ベクトルの使用
q = 非剛体変形パラメータ

画像=> 顔のパーツの散布図モデル, 顔のパーツの分散値λ
顔の形状と表情のアイデンティティーをキャプチャすることが可能
平均形状からの差分λ

face shape (location, scale and rotation)
顔のパーツの場所, 大きさ(全体に占める割合), および回転・傾き
の差分を出すことのできるλ

X = [X1,X2,...,Xn,Y1,Y2,...,Yn,Z1,Z2,...Zn]^T. 4.5
n = 顔の特徴点の数, x,y,zの座標点

この値を使って平均顔(モデルに組み込まれている顔)からの表情動き
